Instructions

1. Search for the Group you would like to analyse into more dept below
2. Click on the corresponding image of the group
3. Wait until all the observations with the corresponding centroids are loaded
4. Once the observations are loaded, click on the observation you want to analyse
5. Wait until the plots are loaded
6. Check the high level summary of the plot on the right hand side
7. If you want to check more detailed plots chose an Image type
8. Chose a step of the observation you want to check
9. Check the more detailed plots with slits below the highlevel plot
10. If you want to analyse a different Image type, chose another Image type
11. If you want to analyse a different step, chose another from the drop down menu
12. If you want to analyse a different observation from the same centroid, just click on another one at the same page
13. If you want to start again, click on "Home" in the menu
